Constructor para mostrar la foto de perfil con o sin símbolo de estar conectado.

Es el constructor genérico para mostrar la foto de perfil.

Las opciones son:

  • La de pantalla de Login con círculo de notificaciones.
  • La de las historias con el borde alrededor.
  • La foto sin ningún elementos por encima.
  • Si no es ninguna de las anteriores, mostrarla con el círculo verde si es que está conectado el usuario.

isPictureFromLogin y isPictureFromStory = false porque en este constructor se muestra la imagen con Online o sin elementos, dependiendo de lo que se envíe en isPictureWithoutElements.

  • Estos ya tienen sus propios constructores.

Los elementos que están después de los dos puntos (:) están siendo inicializados.
