isProfilePictureFromInternet

isProfilePictureFromInternet property

bool isProfilePictureFromInternet
final

Booleano para indicar si la foto de perfil proviene del sistema de archivos o de internet.

  • Si la imagen viene de internet, utilizar CachedNetworkImageProvider.

  • Si la imagen no viene de internet significa que viene de los Assets, por lo que hay que utilizar AssetImage.

El linter marcaba error si el casting se hac√≠a de la siguiente manera:

backgroundImage: isProfilePictureFromInternet
   ? CachedNetworkImageProvider(imageUrl)
   : AssetImage(imageUrl);

Por lo que se tuvo que hacer un cast:

backgroundImage: isProfilePictureFromInternet
   ? CachedNetworkImageProvider(imageUrl)
   : AssetImage(imageUrl) as ImageProvider;
